      • getBestMoveAlt

        public String getBestMoveAlt(int depth,
                             int waitTime)
                      throws IOException
        This function returns the best move for a given position after it has reached a depth of 'depth'. It will check that it has reached depth of 'depth' after 'waitTime' seconds.
        Parameters:
        depth - The depth the engine will go to in the search for the best move
        waitTime - The initial time the engine will wait before checking if analysis to depth 'depth' has been achieved
        Returns:
        Best Move in Long Algebraic Notation format
        Throws:
        IOException - Error on executing UCI command
